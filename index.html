ChuckWagon
A mobile food truck tracking app

ChuckWagon is A Charleston based food truck tracking app currently built for iOS. I worked closely with a front end team to provided a RESTful API using Spring and PostGreSQL to house, filter, maintain, and return data. The front end uses google maps and Ionic.
